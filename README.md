# Прогнозування відтоку клієнтів (Telco Customer Churn)

Цей проєкт присвячений розробці моделі машинного навчання для передбачення відтоку клієнтів телеком-компанії.

## Основні етапи:
1. **Препроцесинг:** Обробка пропусків, чистка даних та кодування категоріальних ознак.
2. **EDA:** Візуальний аналіз кореляцій та факторів впливу.
3. **Моделювання:** Порівняння Logistic Regression та Random Forest.
4. **Результат:** Обрано модель логістичної регресії з Recall 0.79, що є оптимальним для бізнес-задачі утримання клієнтів.

**Стек:** Python (Pandas, Scikit-learn, Seaborn, Matplotlib).
